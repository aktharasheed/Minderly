# Minderly

Minderly is a minimal, data-driven journaling and mood-tracking app built to help users understand their daily thoughts, emotions, and behavioral patterns. It blends a clean user experience with lightweight machine-learning components to surface meaningful insights without overwhelming the user.

Core Features

Fast, structured journaling flow with customizable prompts
Daily mood & habit tracking with timestamped entries
ML-powered pattern detection, identifying trends in mood, habits, and recurring emotional cues
Analytics dashboard for visualizing progress and mental patterns over time
Local-first data handling with optional cloud sync (privacy-focused)
Modular architecture designed for easy expansion of new insight models

Tech Stack

Frontend: React Native
Backend: Node.js
Database: Firestore / MongoDB / Supabase
ML Layer: Lightweight NLP + statistical models for trend detection

Minderly aims to be a calm, intelligent companion that helps users reflect consistently and understand themselves better through meaningful, data-backed insights.
